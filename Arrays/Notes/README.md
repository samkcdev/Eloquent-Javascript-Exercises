## Objects Basics

Objects allows us to map key to values.

eg: <pre>

var employee = {
firstName:"Sam",
lastName:"Cherian",
designation:Developer
}

</pre>

We can access objects values either using the dot notation or Bracket Notation

eg:<pre>
employee.firstName
result will be "Sam"

</pre>
